# Resume Builder

> Resumes that get past the robots.

A 100% client-side, privacy-first resume builder that creates ATS-optimized resumes. No signup required, no data leaves your browser.

**Live at:** [resume.kpruthvi.com](https://resume.kpruthvi.com)

## Features

- **100% Private** - No account needed, all data stays in your browser
- **ATS-Optimized Templates** - Designed to pass Applicant Tracking Systems
- **ATS Score** - Real-time feedback on how ATS-friendly your resume is
- **ATS Preview** - See your resume the way parsers see it
- **Job Description Matcher** - Paste a job description, get keyword suggestions
- **Live Preview** - See changes as you type
- **Dark/Light Mode** - Easy on the eyes

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Zustand (state management)
- jsPDF (PDF generation)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Privacy

This app runs entirely in your browser. Your resume data is stored locally in your browser's localStorage and is never sent to any server.
